# 배포 가이드

## 🚀 빠른 배포 (Netlify 추천)

### 가장 쉬운 방법: Netlify Drop

1. **https://app.netlify.com/drop** 접속
2. 이 폴더 전체를 드래그하여 브라우저에 드롭
3. 자동으로 배포 완료! (URL 즉시 생성)
4. 배포 URL 예시: `https://your-site-name.netlify.app`

## 📝 배포 전 수정할 내용

### ⚠️ 반드시 수정해야 할 링크들

`index.html` 파일을 열어서 다음 내용을 수정하세요:

#### 1. SNS 링크
```html
<!-- 47번째 줄부터 -->
<a href="YOUR_YOUTUBE_LINK" target="_blank">          ← 유튜브 링크
<a href="YOUR_INSTAGRAM_LINK" target="_blank">        ← 인스타그램 링크
<a href="YOUR_TIKTOK_LINK" target="_blank">           ← 틱톡 링크
<a href="YOUR_BLOG_LINK" target="_blank">             ← 네이버 블로그 링크
<a href="YOUR_FACEBOOK_LINK" target="_blank">         ← 페이스북 링크
<a href="mailto:YOUR_EMAIL">                          ← 이메일 주소
<a href="tel:YOUR_PHONE">                             ← 전화번호
```

#### 2. 네이버 클립 링크
```html
<!-- 76번째 줄 -->
<a href="YOUR_NAVER_CLIP_LINK" target="_blank">       ← 네이버 클립 링크
```

#### 3. 비즈니스 제안 이메일
```html
<!-- 68번째 줄 -->
<a href="mailto:YOUR_EMAIL" class="business-button">  ← 이메일 주소
```

#### 4. 쿠팡 파트너스 제품 링크
```html
<!-- 제품 1 - 100번째 줄 -->
<a href="YOUR_COUPANG_LINK_1" target="_blank">        ← 첫 번째 제품 링크

<!-- 제품 2 - 120번째 줄 -->
<a href="YOUR_COUPANG_LINK_2" target="_blank">        ← 두 번째 제품 링크

<!-- 제품 3 - 140번째 줄 -->
<a href="YOUR_COUPANG_LINK_3" target="_blank">        ← 세 번째 제품 링크
```

## 🎨 이미지 교체 방법

### 프로필 이미지
1. 자신의 프로필 이미지를 준비 (정사각형 권장)
2. 이미지 이름을 `profile.jpg` 또는 `profile.png`로 저장
3. `index.html` 32번째 줄 수정:
```html
<img src="profile.jpg" alt="무브먼트 프로필">
```

### 제품 이미지
1. 제품 이미지들을 폴더에 저장 (예: `product1.jpg`, `product2.jpg`)
2. `index.html`에서 각 제품의 이미지 경로 수정:
```html
<img src="product1.jpg" alt="제품명">
```

## 🌐 배포 방법 상세

### 방법 1: Netlify (가장 추천 ⭐⭐⭐)

**장점:** 무료, 초간단, 즉시 배포, HTTPS 자동

1. https://app.netlify.com 회원가입
2. "Add new site" → "Deploy manually" 클릭
3. 폴더를 드래그 앤 드롭
4. 완료! URL 자동 생성

**커스텀 도메인 연결:**
- Site settings → Domain management
- Add custom domain 클릭

---

### 방법 2: GitHub Pages (무료)

**장점:** 무료, 안정적, Git 버전 관리

1. GitHub 계정 생성 (https://github.com)
2. 새 저장소 생성 (저장소 이름: `moovlog`)
3. 터미널에서 실행:

```bash
cd "c:\Users\7tpwz\OneDrive\문서\무브먼트(moovlog) 페이지"
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/moovlog.git
git push -u origin main
```

4. 저장소 Settings → Pages
5. Source: `main` 브랜치 선택
6. Save → 배포 완료
7. URL: `https://YOUR_USERNAME.github.io/moovlog/`

---

### 방법 3: Vercel (무료)

**장점:** 빠른 배포, 자동 HTTPS

1. https://vercel.com 회원가입
2. "New Project" 클릭
3. GitHub 연동 또는 폴더 업로드
4. Deploy 클릭
5. URL: `https://moovlog.vercel.app`

## 🆘 문제 해결

### 페이지가 제대로 안 보여요
- 브라우저 캐시 삭제 (Ctrl+F5)
- 개발자 도구(F12) → Console 탭에서 에러 확인

### 링크가 작동하지 않아요
- `index.html`에서 `YOUR_XXXXX_LINK` 부분을 모두 실제 링크로 교체했는지 확인
- 링크 앞에 `https://`가 포함되어 있는지 확인

### 이미지가 안 보여요
- 이미지 파일이 같은 폴더에 있는지 확인
- 이미지 파일명이 `index.html`의 경로와 일치하는지 확인
- 이미지 파일 형식 확인 (jpg, png, gif 권장)
